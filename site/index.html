The same hardened parser, measured the same way (warmup + best of timed trials) on identical payloads, in contiguous and multi-segment modes. C# Ultra is the standalone managed parser; Pure C is the native core; the bindings reach that core from .NET and the JVM. The bindings call the same C, so the native columns are within measurement noise of each other — the meaningful gap is native vs. managed. Native multi-segment is contiguous + a memcpy (no per-call allocation), unlike the managed multi-segment path.
